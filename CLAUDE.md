# Museum Guide App — Project Context

## What this is
A Flutter Android app that acts as a museum audio guide using smart glasses (Raspberry Pi camera). It recognizes paintings via an on-device YOLOv8 ONNX model, sends the result to a Flask backend (GPT-4o-mini + OpenAI TTS), and speaks the answer aloud through the glasses.

## Architecture
```
Raspberry Pi camera → Flutter ONNX recognition → Flask backend → GPT-4o-mini answer → OpenAI TTS → audioplayers MP3
```

## Key files
- `lib/core/config/app_config.dart` — AppConfig singleton with mode enums (LlmMode, RecognitionMode, FrameSourceMode)
- `lib/core/config/llm_config.dart` — API keys, Flask URL, model settings
- `lib/services/backend_service.dart` — Flask API client (/ask, /process endpoints)
- `lib/services/llm_service.dart` — Three-mode LLM: mock / live (direct OpenAI) / flask
- `lib/services/tts_service.dart` — Plays MP3 audioBytes from Flask, or mock delay
- `lib/services/recognition/live_recognition_backend.dart` — YOLOv8 ONNX inference
- `lib/services/recognition/mock_recognition_backend.dart` — Returns real painting IDs for testing
- `lib/data/mock_artifacts.json` — 12 real paintings (matches paintings.json from Flask backend)
- `lib/screens/settings_screen.dart` — Runtime mode switching UI
- `lib/screens/home_screen.dart` — Main screen with Talk button, painting picker, stop/repeat
- `lib/services/speech_service.dart` — Real speech-to-text using device mic
- `lib/services/frame_source/pi_frame_source.dart` — Polls Pi camera HTTP server
- `lib/services/frame_source/url_frame_source.dart` — Generic HTTP JPEG poller
- `lib/widgets/talk_button.dart` — Animated pulsing button (StatefulWidget)
- `lib/widgets/answer_card.dart` — Answer display with Stop/Repeat buttons
- `lib/widgets/artifact_card.dart` — Shows painting name, confidence, and photo if available
- `~/camera_stream.py` (on Pi) — picamera2 HTTP server, serves JPEG at port 8080

- File: `~/Desktop/best.onnx` → copy to `assets/models/best.onnx`
- Input: 640×640 RGB float32 NCHW, normalized 0–1
- Output: `[1, 16, 8400]` — YOLOv8 format, 12 painting classes + 4 bbox coords
